# 格式方針

#### 排版 \(Layout\) {#排版-layout}

> **堅持** 每個區塊、清單項目、表格元素都置於新行。

```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<table>
  <tbody>
    <tr>
      <td>姓名</td>
      <td>性別</td>
    </tr>
  </tbody>
</table>
```

#### 空格 \(Spacing\) {#空格-spacing}

> **堅持 **Tab 設為 2 spaces。

```html
<html>
  <head></head>
  <body></body>
<html>
```
